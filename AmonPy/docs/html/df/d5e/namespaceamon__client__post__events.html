Detailed Description

client that sends events to the server using HTTP protocol with method=POST

client that sends events to the server using HTTP protocol with method=POST Run with twistd.

Default is run as a daemon process. twistd -l client.log –pidfile client.pid clientpost –hostport "http://127.0.0.1:8000" –epath /path_to_client_ents Kill it with kill cat client.pid Before running make directory archive within directory where your client events live. Each sent event will be moved from client directory to archive directory. Modify if you do not want to save sent events.
