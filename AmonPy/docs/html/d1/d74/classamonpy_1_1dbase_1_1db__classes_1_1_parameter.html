Detailed Description

Creates the Parameter class.

Instances are created by specifying a unique four of IDs (name, event_eventStreaemConfig_stream, event_id, event_rev). The other data attributes are set to defaults on init. This style may later be replaced by the class factory and slots approach, currently adopted for the Event class (see above).
