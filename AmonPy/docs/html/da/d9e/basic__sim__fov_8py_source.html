3 # Generates a relatively simple simulation of sme high energy observatories
4 # The following crude assumptions are made:
5 # 1. Circular FOV
6 # 2. Constant event rate across FOV
7 # 3. Fixed error for PSF
8 # 4. Observatory is fixed to the Earth and cannot change its pointing
9 # 5. No skymap is used (i.e. PSF is analytic)
10 # 6. Run simulation with injected signal if desired with function signal_inject
11 # 7. Calculates FOV overlap
