\subsection{Detailed Description}
client that sends events to the server using H\-T\-T\-P protocol with method=P\-O\-S\-T client that sends events to the server using H\-T\-T\-P protocol with method=P\-O\-S\-T Run with twistd.

Default is run as a daemon process. twistd -\/l client.\-log --pidfile client.\-pid clientpost --hostport \char`\"{}http\-://127.\-0.\-0.\-1\-:8000\char`\"{} --epath /path\-\_\-to\-\_\-client\-\_\-ents Kill it with kill {\ttfamily cat client.\-pid} Before running make directory archive within directory where your client events live. Each sent event will be moved from client directory to archive directory. Modify if you do not want to save sent events.
