\subsection{Detailed Description}
Class to store all of the information needed by the My\-S\-Q\-L database for a set of observations at a given time. 

Information will be stored in tuples, and each observation in a given detection will have its own tuple element i.\-e. if there are two observations at the same time, the time tuple will be a 2 element tuple of identical times
